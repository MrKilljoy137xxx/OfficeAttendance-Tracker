## 1. Project Objectives

The primary objective of this project is to develop a web-based attendance tracker that enables employees to check in and out efficiently, while providing administrators with real-time attendance data and reporting capabilities. This supports the organization's goal of improving operational transparency and reducing manual record-keeping.

## 2. Planning Methodology

The project follows an Agile methodology, utilizing weekly sprints and a Kanban board to manage tasks. This iterative approach allows for continuous feedback, rapid prototyping, and adaptive planning. Each sprint focuses on delivering incremental improvements and functional components of the application.
